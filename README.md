## CNV_PARA_O_20
##
### Instructions

Compile everything:
```bash
$ make all
```
Individual Modules: (some may have dependencies)
Compile __mazerunner__
```bash
$ make maze
```
Compile __bit__
```bash
$ make bit
```
Compile __loadbalancer__
```bash
$ make loadbalancer
```
Compile __storage__
```bash
$ make storage
```
Compile __httpserver__
```bash
$ make httpserver
```
Compile __instrumentation__
```bash
$ make instrumentation
```
___

Clean everything:
```bash
$ make clean
```
___

Refresh environment:
```bash
$ make refresh
```
___

__Instrument__ mazerunner:
```bash
$ make run_inst
```
___

Run __httpserver__:
```bash
$ make run_httpserver
```

___

Run __load balancer__:
```bash
$ make run_balancer
```
